Loan Approval Prediction with Deep Learning

Problem: Predict whether a loan application should be approved based on applicant profile and financial attributes, using a neural network on structured/tabular data.

Dataset: Loan eligibility dataset containing features such as gender, marital status, dependents, education, employment status, income, loan amount, credit history, and property area.

Approach:

  • Performed data cleaning: handled missing values, encoded categorical variables, and standardized numeric features.
  • Exploratory data analysis (EDA) to understand class balance and feature distributions.
  • Built a feed-forward deep neural network (ANN) in Python for binary classification of loan status.
  • Used train/validation split and monitored performance via accuracy and loss curves.
  • Compared deep learning performance with baseline ML approaches to understand trade-offs.

Key Outcome:

  • Identified key drivers of loan approval such as credit history, income, and loan amount.
  • Demonstrated how deep learning can be applied to structured business data, not just images or text.

Titanic Survival Prediction

Problem: Predict passenger survival probability based on socio-economic and personal attributes (age, class, sex, fare, family).

Dataset: Titanic dataset with passenger demographics, class, fare, and related features.

Approach:

  • Performed data cleaning and preprocessing (handling missing values, encoding categorical features, scaling numerical variables).
  • Conducted Exploratory Data Analysis (EDA) to understand survival patterns and feature importance.
  • Trained classical ML models (Decision Tree / Random Forest) and evaluated their performance.
  • Evaluated models using accuracy, confusion matrix, and compared performance across algorithms.

Key Outcome:

  • Identified key predictors of survival such as passenger class, sex, and fare.
  • Demonstrated how well-tuned classical ML models perform on structured data predicting human-centric outcomes.
